# Claude Handoff

This is the cleaned Bitegit yellow project folder for frontend + backend work.

Run:

```bash
npm install
PORT=3000 npm start
```

Main frontend files:

- `public/index.html`
- `public/script.js`
- `public/styles.css`
- `public/coinflare.css`
- `public/mobile-nav.css`
- `public/auth.html`
- `public/auth.js`
- `public/auth.css`

Main backend files:

- `server.js`
- `index.js`
- `routes/auth.js`
- `routes/p2p-orders.js`
- `lib/db.js`
- `lib/repositories.js`
- `middleware/`
- `services/`

Important notes:

- This folder was prepared as a simple handoff folder.
- `node_modules` is not included.
- Environment values are in `.env`.
